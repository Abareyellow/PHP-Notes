PHP
- Powerful, free, open-source programming language
- Widely used in the web development industry
- Fast, Stable, Cross-Platform
- PHP can run on Linux, Windows, & MAC
- Testing Server can be installed using WAMP/MAMP
- PHP Processes scripts on a Web Server
- PHP is a server side language
- JavaScript, HTML, CSS are client side languages

PHP Variables Name Rules:
- Must start with a $ sign, follow by the rest of the Name
- then have a letter or underscore
- Can't start with a number
- Only contain alpha-numeric characters and underscores
- Variable names are case-sensitive
